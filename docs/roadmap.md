# Research Roadmap

## Milestone 1: Baseline Simulator

- Implement validated UWOC channel models.
- Add environment presets for clear ocean, coastal water, harbor water, river, and lake scenarios.
- Provide reproducible baseline experiments.
- Add unit tests and example scripts.

## Milestone 2: AI-Ready Dataset Generation

- Generate synthetic channel datasets.
- Export data in reproducible formats.
- Add train/test scenario splits.
- Document dataset assumptions and physical parameters.

## Milestone 3: Learning-Based Link Adaptation

- Implement supervised channel-state prediction.
- Add reinforcement learning for power control and modulation selection.
- Compare AI-powered adaptation against classical baselines.

## Milestone 4: Digital Twin for UWOC

- Implement digital twin state representation.
- Synchronize simulated and observed data.
- Support predictive analysis and transfer learning.
- Add digital twin-assisted experiment examples.

## Milestone 5: Network-Level Intelligence

- Add multi-node UWOC network models.
- Implement routing and resource allocation baselines.
- Introduce graph neural networks for topology-aware optimization.

## Milestone 6: Publication-Quality Benchmarks

- Define benchmark scenarios.
- Add experiment tracking.
- Produce reproducible figures and tables.
- Prepare manuscript-ready results.

## Candidate Publications

1. OpenUWOC-AI: An Open Research Framework for AI-Powered Underwater Wireless Optical Communications.
2. Digital Twin-Assisted Channel Prediction for UWOC Systems.
3. Reinforcement Learning for Energy-Efficient UWOC Link Adaptation.
4. Generative AI for Synthetic Underwater Optical Channel Modeling.
5. Graph Neural Networks for Intelligent UWOC Network Optimization.
