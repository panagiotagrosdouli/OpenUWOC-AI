# Roadmap

## Implemented

- UWOC attenuation/noise/pointing/turbulence simulation scaffolds.
- OOK modulation and threshold detection.
- Deterministic end-to-end simulation.
- BER/SER/SNR/confidence metrics.
- YAML experiment runner and plotting utilities.

## Prototype

- Neural equalizer and BER predictor.
- Demo GIF generation from code.
- Simulation-only adaptive decision proxy.

## Planned

- Literature-calibrated water optical coefficient database.
- Multipath channel impulse response.
- BPSK, QPSK, M-QAM, and OFDM stacks.
- Forward-error-correction module.
- Matched filter and linear equalizer baselines.
- Dataset versioning for tank/field experiments.
- Adaptive modulation policy trained and evaluated against baselines.
- Robotics integration for AUV/ROV communication scenarios.
