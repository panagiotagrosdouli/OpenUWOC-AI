1. A clean Python package architecture for simulation-first AI-assisted UWOC research.
2. Physics-inspired baseline channel components covering absorption, scattering, Beer-Lambert attenuation, pointing loss, turbulence scaffold, thermal noise, shot-noise approximation, and ambient light.
