# Abstract Scaffold

Underwater optical wireless communication (UWOC) can provide high-rate short-range links for marine robotics and underwater sensing, but performance is limited by absorption, scattering, turbulence, misalignment, and optical/electrical noise. This repository introduces OpenUWOC-AI, a reproducible simulation-first software framework for evaluating classical and AI-assisted physical-layer UWOC receivers. The current implementation includes Beer-Lambert attenuation, water-type simulation presets, pointing-error loss, thermal and shot-noise models, OOK modulation, threshold-detection baselines, BER/SER/SNR metrics, YAML-configured experiments, and prototype neural equalizer and BER-prediction models. The framework is intended to support future measured-data validation, adaptive modulation, and robust receiver learning. Current results are simulation-only and do not claim state-of-the-art performance.
