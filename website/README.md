# OpenUWOC-AI Website

This directory is reserved for the research project website.

## Planned stack

- Next.js
- TailwindCSS
- Framer Motion

## Required sections

- Hero section with the central UWOC-AI research question.
- UWOC system diagram.
- AI receiver diagram.
- Code-generated demo GIF from `assets/demo.gif`.
- Simulation-only results and limitations.
- Roadmap.
- GitHub and citation links.

The website should not display unverified benchmark numbers or state-of-the-art claims.
